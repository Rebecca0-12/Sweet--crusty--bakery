# Sweet Crust Bakery — Business Landing Page

A modern, elegant single-page website concept designed for an artisan bakery business. Built as a client-style project to showcase a warm, editorial brand aesthetic for a small food business — from hero storytelling to menu, gallery, testimonials, and a contact form.

**🔗 Live Demo:** [view the website](https://shiny-malasada-f40a0f.netlify.app)

## About This Project

This was designed as a complete front-end concept for a bakery business, focused on giving a small, local brand a premium, boutique-quality web presence. The goal was to make the site feel handcrafted — much like the product itself — using warm typography, a rich brown-and-gold palette, and generous whitespace instead of generic templated layouts.

## Features

- **Animated hero section** with staggered fade-in entrance and scroll indicator
- **Our Story section** with stats counters (years baking, recipes, happy customers)
- **Signature menu grid** with hover interactions and pricing
- **Responsive photo gallery** with an asymmetric bento-style layout
- **Customer testimonials** section with star ratings
- **Contact section** with business info and a styled inquiry form
- **Scroll-reveal animations** using the Intersection Observer API
- **Shrinking sticky navbar** on scroll
- Fully responsive across desktop, tablet, and mobile breakpoints

## Tech Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (Intersection Observer, scroll events)
- Google Fonts — [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) & [Jost](https://fonts.google.com/specimen/Jost)

No frameworks, no build tools — a single self-contained HTML file, deployed as a static site.

## Running Locally

Clone the repo and open the file directly, or serve it locally:

```bash
git clone https://github.com/Rebecca0-12/sweet-crust-bakery.git
cd sweet-crust-bakery
# then just open index.html in your browser
```
